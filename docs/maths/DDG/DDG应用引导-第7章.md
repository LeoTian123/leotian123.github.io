# 第7章 曲面共形参数化

## 1 基本思想回顾

目的是想办法把一个立体mesh映射到平面上，就像是绘制地图那样。这个映射必然损失一些事情，所以这里考虑的是“保留了一些事情”的某种映射，即**共形映射/保角映射/conformal mapping**。

怎么表达共形映射？好在复变函数给出了很多理论，复变函数 $z$ 是共形映射，那么它必然满足**柯西黎曼方程**。柯西黎曼方程方程的一种表达方式是：

$dz(\mathcal{J}X) = idz(X)$

$\mathcal{J}$ 作用在 $X$ 说的是把它正向（逆时针）旋转90度的那个向量。这里使用这个符号是为了规避出现把一个复数 $i$ 乘在向量上的情况，这看上去很美，但是需要额外的说明，所以不这样。

那么这个 $d$ 依然表示了切向量——理解来看，它作用在一个复变函数 $z$ 上，正像是它作用在一个到空间中的映射 $f$ 上：

![图片](https://pica.zhimg.com/v2-2056cd66a4e4a002d18350261e7fbce0.jpg)

那么有了 $X$ 映射之后的切向量 $dz(X)$ ，还有旋转某个角的 $\mathcal{J}X$ 映射之后的切向量 $dz(\mathcal{J}X)$ ，希望两个切向量还保持这个角的关系，即 $dz(\mathcal{J}X) = idz(X)$ ，所谓保角就是“**旋转再映射等于映射再旋转**”。保持了某个角如此（90度），其他角也是如此，这就是柯西黎曼方程。

一般柯西黎曼方程还会这么写，对于 $f(z)=u(x,y)+iv(x,y),z=x+iy$ ，写：

$\frac{\partial u}{\partial x} = \frac{\partial v}{\partial y}, \quad \frac{\partial u}{\partial y} = -\frac{\partial v}{\partial x}$

那么这个形式和上面是等价的。换成矩阵的语言可以这样证明：

![图片](https://picx.zhimg.com/v2-d1ad4dca4a5840b2ae1d785bd666dac3.jpg)

![图片](https://pic4.zhimg.com/v2-fd90ee2d05abd15ddad733c69b13fd3b.jpg)

在这四个矩阵里，左边相等推出右边相等，右边相等推出左边相等，得证。

这个形式是讨论**复变函数可导性**推导出来的一个性质，所以这个形式更加常见。也就是说，前面那个柯西黎曼方程就是一般提到的柯西黎曼方程。

顺便一提，这个性质加上额外的条件就是可导的充要条件。那么这里，对于一个点，一般这样去讲，从最宽松到最严格：

-   某一点满足柯西黎曼方程：最宽松，讨论的基础
-   某一点是可导的 $\Leftrightarrow$ 导数的极限定义是存在的 意味着 各个方向逼近该点极限都存在且一致 $\Leftrightarrow$ 满足柯西黎曼方程且**u,v可微**（像是多元函数论那样的实变的可微）
-   某一点是解析的/全纯的 $\Leftrightarrow$ 这一点小邻域内可导
-   某一点是共形映射 $\Leftrightarrow$ 这一点解析，且**导数不为零**

就是说，**共形映射等价于导数不为 0 的解析函数**。以上就是柯西黎曼方程与共形映射的事情。

接下来有点夸张，就不细致说明了，首先这个方程放在外代数里可以这样写，回顾霍奇星的意义：$\star dz = idz$ 那么我们就可以优化这个损失：

$EC(z) := \frac{1}{4} || \star dz − idz|| ^2$ 

接下来四个练习题是在化简这个东西：

![图片](https://pic2.zhimg.com/v2-d9e334fa8667ffc9d1229672b7883035.jpg)

![图片](https://picx.zhimg.com/v2-b91e9bbbc2ff5b8b4b190c7fafc3b44d.jpg)

那么我们简化问题，只考虑封闭曲面，没有后面那一项了。前面那一项叫做**狄利克雷能量**，加上练习7.6的结论，最后就是优化 $E_D (z)=z^∗ Lz$ ，其中 $L$ 是拉普拉斯矩阵，就是把那个 $\Delta$ 的形式变为矩阵的形式。

还要加上两个约束：

![图片](https://pic1.zhimg.com/v2-32ea23fed00f5720e6a3fb5f3715a954.jpg)

总问题就是：

$\min_{z}  \frac{1}{2} z^* L z \\ s.t. \ \  z^* z = 1 \\ \ \ \ \  \ \ \ \ \ z^* 1 = 0$

那么这个问题一下子就变得和**谱图理论**中的**谱布局**非常像，也和**瑞利商、广义特征值**问题联系起来了。再多的我也不细说了，不是这里主题，总之这东西结论就是，把 $L$ 的第二大、第三大的特征向量取出来，拼起来得到 $v_2 + iv_3$ ，这就是这个优化问题的解析解，同时也是我们要的共形映射的坐标。

## 2 代码与结果

所以这东西代码没什么好写了，两句话搞定了：

```python
from scipy.sparse.linalg import eigsh

L = build_laplacian_matrix(verts, adj_mat)

evals, evecs = eigsh(L, k=4, sigma=0, which='LM')
u = evecs[:, 1]
v = evecs[:, 2]
```

## 3 延伸思考

我提到了谱布局，那么这个东西和图论当然是太像了，有深刻的联系。

谱布局，把图的顶点看成弹簧，拉普拉斯矩阵描述拉力——由谁给。假设考虑的是带权拉普拉斯矩阵就更好了，权越大，拉力越强，除了“由谁给”还有“给多大”。谱布局是在优化这个力学系统的能量函数：

![图片](https://pic2.zhimg.com/v2-4ecc4e2d1a448b3b50b0ff517b8bbadd.jpg)

权越大，谱布局中，两个点越近，这可以从优化目标看出来。权越小，越接近于没有这个边，越远。

曲面共形参数化，也可以看做一个“力学”系统，把网格看成弹性薄壳，不过这里的“力”是角度结构提供的。我们知道狄利克雷能量 $z^* L z$ 展开了和上面形式一样的，权是cotan定理提供的，“力”就和角结构有关了。于是这两件事就统一了。

进一步类比也是可以的，离得近的点通常对应的两个角更小，cotan更大，权更大。
