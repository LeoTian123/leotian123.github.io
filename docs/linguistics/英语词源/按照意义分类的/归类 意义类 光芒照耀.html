  • PIE*bhel1 照耀 闪光 燃烧
    • 生物
      • beluga/bə'luːgə/白鲟鱼：词源同blank, 白的。形容该鱼的颜色。
      • flamingo/flə'mɪŋgəʊ/火烈鸟：来自flame, 燃烧，炫丽。因这种鸟火红的翅膀而得名。
      • phlox/flɒks/福禄考（高大开花植物）：来自希腊语phlox,火焰，如火焰般绚烂的花，来自phlegein,燃烧，发热，来自PIE*bhel,照耀，燃烧，词源同blaze,bleach.
    • 颜色
      • black/blæk/黑的：来自PIE * bhel, 燃烧，发光。词源同blank, blue. 在古英语里black是一个词义不明确的词，既指黑的，也指白的，或其它颜色。
      • blue/bluː/蓝色的：来自PIE * bhel, 燃烧，发光。词源同blank, black. 在古英语里这一PIE词表示各种颜色。
    • 漂白
      • blanch/blɑːn(t)ʃ/漂白：同blank，白的。
      • bleach/bliːtʃ/漂白：词源同blanch, 漂白。
      • blemish/'blemɪʃ/斑点：词源同bleach, 漂白，白斑，或其它斑点。
    • 物品状态
      • bleak/bliːk/暗淡的，阴冷的；荒凉的：词源同bleach, 漂白，指苍白的，暗淡的。
      • blank/blæŋk/空白的：来自PIE * bhel, 燃烧，发光。此处指发白光，白色。
      • blend/blend/混合：来自PIE *bhel, 照耀，闪光。指闪光，炫目，障眼法，混合。词源同blank, 白的。
        • blond金发的，白肤碧眼金发的人：法语单词的原意是“介于金色和浅栗色之间的颜色”，这可能解释了“混合”的概念
        • blonde 金发女郎：词源同blend, 混合。指金色和棕色的混合颜色。
      • blind/blaɪnd/av瞎的：古英语中的 blind 意为“没有视力”，同时也指“黑暗的，被黑暗所笼罩的，模糊的； 愚昧的，缺乏心智感知”，可能源自原始日耳曼语中的 *blindaz “盲目的”。最初的意义并非“无视力”，而是“困惑的”。
        • blindfold/'blaɪn(d)fəʊld/蒙住眼睛，眼罩：改写自古英语blindfellen。blind, 瞎。fell, 砍伐，击倒。原指打瞎，后改成fold, 折叠，指眼罩。
        • blunder/'blʌndə/nv(因无知、粗心等造成)的错误，踉跄地走：14世纪中叶，“盲目地跌跌撞撞地走”，源自斯堪的纳维亚语，类似于古诺尔斯语 blundra “闭上眼睛”，可能来自原始日耳曼语 *blinda- “盲”。
          • blunt/blʌnt/钝的，不锋利的；生硬的；直率的：约1200年， blunt， blont，“钝的，迟钝的”（用于人），起源不明。可能来自或与古诺尔斯语 blundra “闭上眼睛”。
      • fulgent/'fʌldʒ(ə)nt/光辉的，灿烂的：来自PIE*bhel, 照明，照耀，词源同blaze, fulminate.
        • refulgent/rɪ'fʌldʒ(ə)nt/灿烂的：re-,向后，往回，-fulg,燃烧，发光，词源同 blaze,fulgent.
    • 人状态
      • blench/blen(t)ʃ/惊悸：词源同bleach, 漂白。指由于惊悸而脸色苍白。
      • blush/blʌʃ/脸红的：来自PIE *bhel, 发光，闪耀。指脸发红。词源同blaze, 明亮。
      • flagrant/'fleɪgr(ə)nt/罪恶昭彰的：来自PIE*bhleg, 照明，闪光，来自PIE*bhel的扩大形式，词源同flame, blaze. 原义为闪光的，辉煌的。然后由于不知道的原因词义贬义化。参照中国反腐案例。
      • flamboyant/flæm'bɒɪənt/炫耀的：来自古法语flamboiier, 闪光，照耀，词源同flame. 拼写受boy影响俗化。
    • 雷声
      • fulminant adj. （疾病）爆发性的：fulmin 雷声 + ant …的，具有…性质的 → 像雷声一样爆发的 → 爆发性的
      • fulminate v. 大声斥责：fulmin 雷声 + ate 做，造成，使… → 如雷声般叫喊 → 大声斥责
      • fulmination n. 闪电；爆发；爆鸣；严厉谴责：fulmin 雷声 + ation 行为，过程，状态，结果 → 闪电；爆发；爆鸣；严厉谴责
      • fulminic adj. 爆炸的：fulmin 雷声 + ic …的 → 像雷声一样巨响 → 爆炸的
      • fulminous adj. 雷电的；怒喝的：fulmin 雷声 + ous 有…性质的，关于…的 → 雷电的；怒喝的
    • 燃烧
      • blaze/bleɪz/燃烧：来自PIE *bhel, 燃烧，发光，同blank.
      • conflagration/,kɒnflə'greɪʃ(ə)n/大火：con-, 强调。-flag, 火，燃烧，词源同blame, flagrant.
      • flame/fleɪm/火焰：来自PIE*bhel, 燃烧，闪光，词源同blank, blaze.
        • inflame/ɪn'fleɪm/燃烧，激怒：in-,进入，使，flame,燃烧。引申词义使激怒。
        • inflammable/ɪn'flæməb(ə)l/易燃的，易激怒的：来自inflame,燃烧，激怒。引申词义易燃的，易激怒的。
        • inflammation 炎症
        • inflammatory 发炎的 刺激性的 煽动性的
        • flammable/'flæməb(ə)l/易燃的：来自flame, 燃烧。
    • 其他名词
      • blancmange/blə'mɒnʒ; -'mɑːnʒ/果味牛奶冻：blanc, 白色。mange, 吃，见manger, 马嘈。
      • blanket/'blæŋkɪt/毯子：blank, 白的。-et, 小词后缀。原指一块白布，后指白毯子，毯子。
      • blitzkrieg/'blɪtskriːg/闪电战：来自德语，最早出现于1939年德国对波兰的闪电攻袭。blitz, 闪电，来自PIE *bhel, 照耀，闪光，词源同blank. krieg, 战争。
      • flamingo/flə'mɪŋgəʊ/弗拉门科舞（一种节奏明快的西班牙舞蹈）：来自西班牙语。一种观点认为来自Fleming, 民族名，民族舞蹈。或来自flame, 燃烧，炫丽，形容这种舞蹈的欢快。
      • phlegm/flem/痰，冷淡，迟钝，冷静，镇定：来自拉丁语phlegma,痰，黏液（古医学体液理论构成人体的血液，黏液，胆汁，黑胆汁四种体液之一），来自希腊语phlegma,发热，由体热所致的多痰症，来自phlegein,燃烧，发热，来自PIE*bhel,照耀，燃烧，词源同blaze,bleach.后在中世纪医学理论认为体内多痰会致人冷淡，迟钝，但是，被误解为冷淡是冷静，镇定的表现，因此，产生模糊和矛盾的词义。
        • phlegmatic/fleg'mætɪk/冷淡的，迟钝的，冷静的，镇定的：来自phlegm,冷淡，迟钝，冷静，镇定。

      • gall/gɔːl/苦胆，恼怒：来自PIE*ghel, 发光，照耀，词源同yellow, gold, chlorine. 因胆呈黄绿色而得名。引申义恼怒。

      • melancholy/'melənkəlɪ/忧郁，悲伤：来自拉丁语melancholia,来自希腊语malankholia,黑胆汁过度分泌， 忧郁，悲伤，来自melas,黑的，词源同melanin,khole,胆汁，词源同gall.在古代观念中，人的忧郁，悲伤等是由于黑胆汁的过度分泌引起的。参照humor,体液，体液学说。

      • sycophant/'sɪkəfænt/告密者，诽谤者，谄媚者，拍马者：来自拉丁语 sycopanta,来自希腊语 sykophantes,告密者，诽谤者，字面意思为展示无花果的人， 来自 sykon,无花果，-phant,显现，展示。机翻：“展示无花果”是一种粗俗的手势，将拇指插在两个手指之间，这种手势隐约类似于无花果，本身就是阴道的象征（sykon也有“外阴”的意思）。现代公认的解释是，古希腊的著名政治家对这种煽动性的姿态敬而远之，但私下里敦促他们的追随者嘲弄对手。

      • phantasm/'fæntæzəm/鬼魂，幽灵，幻觉，幻影：来自拉丁语phantasma,幽灵，鬼魂，来自希腊语phantasma,影象，幻象，来自phainein,出现，来自PIE*bha,照明，光照，词源同beacon,phase.
        • phantasmagoria/,fæntæzmə'ɡɔrɪə/幻灯，幻觉效应：来自希腊语phantasma,影象，幻象，agora,集会，广场。比喻用法，用于指早期的舞台表演的幻灯，引申词义义幻觉效应。
        • phantom/'fæntəm/鬼魂，幽灵：来自拉丁语phantasma,幽灵，鬼魂，词源同phantasm.
    • phenology/fɪ'nɑlədʒi/物候学：pheno-,显示，展现，-logy,学说。即研究一年四季自然变化的学说。
      • phenomenon/fə'nɑmɪnən/现象，非凡的人：pheno-,显示，展现，-men,名词后缀。引申词义现象，有代表性的体现，后也用于指非凡的人。其复数形式为phenomena.
      • phenotype 表现型的：pheno-,显示，展现，type,类型。比喻用法。

      • photogenic/,foto'dʒɛnɪk/上相的，上镜的：phot-,光，-gen,产生，词源同generate.字面意思即由光产生的，为photography的早期术语，原称为photogenic drawing.后20世纪初照相流行起来后，该词被赋予新义上相的，上镜的。
      • photograph/'fotəɡræf/照片，照相：phot-,光，-graph,写，记录。即把光记录下来，引申词义照相，相片。

      • photosynthesis/,foto'sɪnθəsɪs/光合作用：phot-,光，synthesis,合成作用。
      • phosphorus/'fɑsfərəs/磷：来自拉丁语phosphorus,启明星，来自希腊语Phosphoros,启明星，来自phos,光，phoros,携带者，词源同bring.后用作化学物质磷。

      • phase/fez/月相，相位，阶段：来自希腊语phasis,月相，星相，来自phainein,出现，显示，来自PIE*bha,照明，光照，词源同beacon,phantasm.引申词义相位，阶段。

      • dial/'daɪəl/表盘：来自拉丁短语rota dialis, daily wheel (rotate diurnal )，一种古代测太阳运转的日晷，后用来指相关圆形物体。
      • diary/'daɪəri/日记：来自拉丁语dies, 白天，词源同diurnal. 即一日记录。
      • diet/'daɪət/饮食：来自拉丁语dieta, 一日的工作，饮食，作息。
      • dismal/'dɪzməl/凄凉的，阴沉的：来自拉丁语dies mali, 即malicious day. 后用做形容词，凄凉的，阴沉的。
      • diurnal/daɪ'ɝnl/昼行的：来自拉丁语dies, 白天，来自PIE*dyeu, 照耀，词源同diva, divine,Zeus.
      • journal/'dʒɝnl/日志，期刊，新闻：来自古法语jornel,天，一天时间，一天的工作，来自拉丁语diurnalis,一天的，词源同daily.部分学者认为其拼写变化由字母d到字母j可能是来自后拉丁语时期，字母i变成字母j后导致首字母d脱落造成的。词义由一天的工作引申日志，日报，期刊，一天的新闻等。
      • journalist/'dʒɝnəlɪst/新闻工作者：来自journal,新闻，-ist,人。即从事新闻工作的人。
      • journey/'dʒɝni/长途旅程，行程：来自古法语journee,一天的行程或工作，来自拉丁语diurnum,天，一天，词源同diurnal.后词义不再受一天的行程所局限，引申词义行程，长途旅程。
      • meridian/mə'rɪdɪən/子午线，经线：来自拉丁语meridies,中午，来自medius,中间的，词源同middle,字母d被r音化，dies,天，词源同diurnal.引申词义中午的太阳，并用于地理学术语子午线，经线。
      • quotidian/kwo'tɪdɪən/普通的，日常的：来自拉丁语quotidianus,日常的，每天的，来自拉丁语quot,多少，来自PIE*kwo,疑问代词词干，词源同quality,quantity.-dian,天，词源同diurnal.
      • sojourn/'soʊdʒɝn/逗留，暂住：来自古法语 sojurner,逗留，暂住一段时间，来自通俗拉丁语*subdiurnare,停留一天，玩一天， 来自 sub-,在下，持续，-diurn,天，一天，词源同 diurnal,journal.
    • 照耀
      • psychedelic/'saɪkə'dɛlɪk/迷幻药：药品名，根据希腊语创造出来的一个词。来自希腊语psyche,灵魂，精神，delos,清晰的，可见的，来自PIE*dyeu,照亮，照明，神，词源同diurnal,divine.引申词义产生幻觉的。
