      • pubis/'pjuːbɪs/耻骨：来自1590年代的拉丁语 pubes（属格 pubis）"生殖区域，腹股沟"，通过 os pubis "耻骨"。拉丁语 pubes（n.）与 pubes（adj.）"成年人，成熟的，男子气概的"相关或相同，这个词的起源不确定。
      • pubes/ˈpju:bi:z/阴毛：源自拉丁语 pubes “青春期的，到达青春期的，成熟的，成年的”，也作为名词，“青春期的标志”（如阴毛），还有“青春期的年轻男子”（见 pubis）。在19世纪还指“耻骨”，更早的时候指“构成骨盆前部的任何一侧髋骨的部分”，源自拉丁语 os pubis，来自 pubes “生殖区域”。在现代俚语中，这个单音节词是 pubic hairs 的常见缩写。
      • pubic/'pjuːbɪk/阴毛的,阴部的,耻骨的：1811年，由 -ic 和医学拉丁语 pubis（1590年代）组成，缩写自拉丁语 os pubis，来自拉丁语 pubes（属格 pubis）"生殖区，腹股沟"，与 pubis 相关的 pubes（形容词）"成年的"（见 pubis）。
