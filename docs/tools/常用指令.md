# 常用指令

这里主要是方便平时懒着打字，直接复制指令用的。

## 网络

对于代理软件 v2rayN，要让 CMD 唤起的网络工具也走代理，使用如下指令：

```powershell
set http_proxy=http://127.0.0.1:10808
set https_proxy=http://127.0.0.1:10808
```

ping 命令使用的是 IP 层的 ICMP 协议，所以，以上两个命令设置好，想要测试是不是真能用，应该使用：

```powershell
curl -v https://www.google..com.hk
```

--------------

下载 huggingface 模型时，要么使用上边方法走代理，要么设置环境变量。

设置环境变量的话，不建议在.py里设置，甚至专门写一个.py脚本搞下载也不建议在其中设置，应该在 bash 中设置：

```bash
export HF_ENDPOINT="https://hf-mirror.com"
```

## Git 与 GitHub

如果之前已经跟踪了某个文件，写入.gitignore后还没完，还需要使用“

```bash
git rm --cached <file>
```

也有大规模调整，以至于很多需要取消的，这时候可以全部清掉：

```bash
git rm -r --cached .
```

为了查看当前那些正要提交（正在被跟踪），使用：

```bash
git status
```

这顺便可以看所在的分支、同步的远程分支。

分支操作：

| 指令 | 说明 |
|------|------|
| `git branch` | 列出所有本地分支 |
| `git branch <name>` | 创建新分支 |
| `git checkout <branch>` | 切换到指定分支 |
| `git switch <branch>` | 切换分支（较新方式） |
| `git merge <branch>` | 合并指定分支到当前分支 |
| `git rebase <branch>` | 变基，将当前分支接在目标分支之后 |
| `git branch -d <branch>` | 删除已合并的分支 |
| `git branch -D <branch>` | 强制删除分支 |

添加远程仓库，HTTP版本如下：

```bash
git remote add origin https://github.com/用户名/仓库名.git
```

SSH版本如下：

```bash
git remote set-url origin git@github.com:用户名/仓库名.git
```

push有一个 `-u` 选项，意思是“将本地分支与远程分支关联起来”，以后就可以直接 `git push` 了。也可以不用，每次都完整打出四个单词。

这东西是有撤销与回退的，用到再说，很麻烦。

## 一些服务器启动指令

Django：

```bash
python manage.py runserver 0.0.0.0:8000
```

Python自带的工具：

```bash
python -m http.server 8000
``` 

Vue 项目：

```bash
npm run dev
```
