# 离散微分几何工具箱

代码仓库在[ddg-keenan-crane-python-answers](https://github.com/LeoTian123/ddg-keenan-crane-python-answers)

也可以参见[这里](../maths/DDG/index.md)

主要工作：

- 实现资料《DISCRETE DIFFERENTIAL GEOMETRY: AN APPLIED INTRODUCTION》(Keenan Crane)中离散微分几何的核心算法
- 完成曲率计算、余切拉普拉斯算子、拉普拉斯平滑、共形参数化、霍奇分解、生成元与调和基、connection与向量场设计等几何处理算法
- 基于 pyvista 的统一三维可视化接口，支持网格着色、路径可视化、边与对偶边上的 1‑form 可视化以及面上的切向量场显示。
- 模块化封装为可复用工具包，支持 OBJ 模型加载，简洁高效
